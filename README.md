
# hwig

Calculates Half-Weight Index Gregariousness (HWIG) as described in Godde
et al. (2013) [\[1\]](#references)

## Installation

``` r
devtools::install_gitlab('robit.a/hwig')

# CRAN
# install.packages('hwig')
```
